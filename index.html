What is expo

So what exactly is expo?

Expo is built upon react and javascript allowing it to be deployed to all platforms after passing through an interpreter.

Basically - Expo allows you to deploy to multiple platforms with one code

Unsurprisingly, the width controls the width of the element, and height controls the height of the element. This allows them to determine the size of the element.

Top and Left

Top controls how far from the top the element is and left controls how far from the left edge the element is. Now this can be quite complicated and will have more detail below. But this basically allows you to control the position.
